Midterms

The main function is to update Excel files when there are new entries in the On-line Exam Accommodation Request Module:
https://www.mcgill.ca/osd/node/879/webform-results.

Files to be updated depend on the term of the exam. Their names have the following structure: "term exam schedule.xlsx", where term is a semester, in which exams are taken. Example: "Winter 2013 exam schedule.xlsx".

If the required file is not found, it will be created.

As well it updates "Files/rooms_midterm.xlsx" file, allocating available places.

There is an option to download the whole database and to show the last id (id of the last entry in the database after the last update).
